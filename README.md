# Form Builder Frontend

React + TypeScript + Vite ile geliştirilmiş No-Code Form Builder frontend uygulaması.

## Özellikler

- React 18 + TypeScript
- Vite
- Material-UI
- Zustand (State Management)
- React DnD (Drag & Drop)
- React Router DOM
- Axios

## Kurulum

```bash
cd fbuilder-frontend
npm install
npm run dev
```

Uygulama http://localhost:3000 adresinde çalışacaktır.

## Kullanım

1. Login sayfasından giriş yapın (varsayılan: admin/admin123)
2. Forms sayfasından mevcut formları görüntüleyin
3. "Yeni Form" butonuna tıklayarak form builder'ı açın
4. Sol panelden bileşenleri sürükleyip canvas'a bırakın
5. Sağ panelden seçili alanın özelliklerini düzenleyin
6. "Kaydet" butonuna tıklayarak formu kaydedin
7. Form renderer sayfasından formu görüntüleyip veri girişi yapın
